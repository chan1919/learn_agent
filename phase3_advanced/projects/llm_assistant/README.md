# 基于LLM的智能助手项目

## 项目简介

这是一个基于大语言模型的智能助手项目，具备以下功能：
- 多轮对话能力
- 工具使用能力
- 个性化服务
- 任务自动化

## 目录结构

```
llm_assistant/
├── README.md            # 项目说明
├── requirements.txt     # 依赖包
├── config.py            # 配置文件
├── agent.py             # 智能助手核心
├── tools.py             # 工具定义
├── memory.py            # 记忆管理
└── main.py              # 主程序
```

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置API密钥

在 `config.py` 中设置你的API密钥，或设置环境变量：

```bash
export OPENAI_API_KEY=your_api_key
```

### 3. 运行程序

```bash
python main.py
```

## 功能特性

### 1. 多轮对话

智能助手能够保持对话上下文，进行连贯的多轮对话。

### 2. 工具使用

内置多种工具，如：
- 天气查询
- 计算器
- 时间查询
- 文件操作

### 3. 个性化服务

根据用户的偏好提供个性化的服务。

### 4. 任务自动化

能够根据用户的指令自动执行一系列任务。

## 扩展指南

### 添加新工具

在 `tools.py` 中添加新的工具函数，然后在 `agent.py` 中注册。

### 自定义记忆管理

修改 `memory.py` 中的记忆管理逻辑。

### 调整LLM参数

在 `config.py` 中调整LLM的参数，如温度、最大 tokens 等。

## 示例对话

```
用户: 今天天气怎么样？
助手: 请问您想查询哪个城市的天气？
用户: 北京
助手: 北京的天气晴朗，温度25℃

用户: 12345 + 67890 等于多少？
助手: 计算结果: 80235

用户: 帮我创建一个名为 test.txt 的文件，内容是 "Hello World"
助手: 已成功创建文件 test.txt，内容为 "Hello World"
```
